# 🎓 Student Attendance Management System Using Zoho Creator

## 📖 Project Overview
The **Student Attendance Management System Using Zoho Creator** is a cloud-based application developed to automate and streamline the process of recording and managing student attendance in educational institutions. The system replaces traditional manual attendance methods with a digital platform, improving accuracy, accessibility, and efficiency.

This application allows teachers or administrators to mark attendance, manage student records, and monitor attendance data in real time through an easy-to-use interface built on the Zoho Creator platform.

---

## 🎯 Objectives
- Digitize student attendance management  
- Reduce manual errors and paperwork  
- Provide real-time attendance tracking  
- Maintain centralized student records  
- Generate attendance reports  

---

## ⚙️ Features
- Student registration and database management  
- Attendance marking (Present/Absent)  
- Class and subject management  
- Attendance history tracking  
- Report generation  
- Cloud-based access via Zoho Creator  

---

## 🛠️ Technologies Used
- Zoho Creator (Low-code platform)  
- Deluge Script (Zoho scripting language)  
- Cloud Database (Zoho Creator Data Storage)  
- Web-based Interface  

---

## 👨‍🎓 Users
- Teachers / Faculty  
- School or College Administrators  
- Educational Institutions  

---

## 🚀 Advantages
- Paperless attendance management  
- Real-time data access  
- Accurate record keeping  
- Easy monitoring of student attendance  
- Secure cloud storage  

---

## 📌 Conclusion
The Student Attendance Management System developed using Zoho Creator provides an efficient and reliable solution for managing attendance digitally. By automating attendance tracking and reporting, it enhances administrative productivity and ensures accurate student records.
